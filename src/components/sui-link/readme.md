# Link

## Properties

| Property | Type       | Default     | Description             |
| -------- | ---------- | ----------- | ----------------------- |
| href     | `string`   | `undefined` | Link address            |
| variant  | `"button"` | `undefined` | Applies `button` styles |

## How to use

```
<sui-link href="/">Link text</sui-link>
```
